# trade-exec-engine
Takes care of order execution, management &amp; reporting 



## Dev Env Setup Steps
1. Git checkout
2. Run Dev setup 
```commandline
sh scripts/dev-setup.sh
```



## Production Env Setup Steps
1. Install Python 3.10 or higher
2. Git checkout
3. Run Dev setup 
```commandline
sh scripts/prod-setup.sh
```
